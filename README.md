# 🐘 FANTE

**Plataforma Adaptativa de Aprendizaje para Necesidades Educativas Especiales (NEE) y TEA.**

Fante es una aplicación web diseñada para transformar el tiempo de pantalla de los niños en secuencias estructuradas que bajan la ansiedad y construyen habilidades, un día a la vez.

## 📖 Nuestra Historia (El Porqué)
Fante nació de una experiencia real con Vicente, un niño con TEA y retraso madurativo. Descubrimos que respondía extraordinariamente bien a la previsibilidad, la secuencia visible y la recompensa inmediata de aplicaciones gamificadas (logrando rachas de más de 325 días ininterrumpidos). 

Sin embargo, no existía una herramienta que permitiera a los padres y terapeutas **crear contenido a medida** usando esas mismas mecánicas para trabajar objetivos terapéuticos, rutinas o habilidades de la vida diaria. Así nació Fante.

## 🎯 ¿Qué hace Fante?
Fante combina el poder de la **gamificación** (rachas diarias, puntos de experiencia, una mascota amigable) con un **creador de cursos flexible**. 

- **Para los niños (Alumnos):** Es un entorno seguro, predecible y libre de frustraciones. No hay "cruces rojas" ni castigos por equivocarse, solo invitaciones a intentarlo de nuevo. Aprender se siente como un juego. **Siempre será 100% gratuito para ellos.**
- **Para las familias:** Permite crear lecciones personalizadas adaptadas al nivel de sus hijos, o utilizar cursos creados y validados por la comunidad.
- **Para los profesionales (Psicopedagogos/Terapeutas):** Permite extender las terapias más allá del consultorio, asignando lecciones específicas y monitoreando el progreso diario y las métricas de cada paciente.

## ✨ Características Principales
- **Rachas Diarias:** Fomentan la constancia y la creación de rutinas.
- **Micro-aprendizaje:** Lecciones divididas en bloques simples (texto, imagen, video, audio, opciones múltiples) mostrando una sola acción por pantalla para evitar la sobreestimulación.
- **Refuerzo Positivo:** Fante (el elefante) acompaña al niño celebrando sus logros con frases motivacionales.
- **Catálogo Comunitario:** Un espacio para compartir, explorar y clonar cursos entre familias y profesionales.

## 🛠️ Stack Tecnológico
- **Frontend:** React + TypeScript + Vite
- **Estilos:** Tailwind CSS + shadcn/ui
- **Backend, Base de Datos y Autenticación:** Supabase (PostgreSQL)
- **Arquitectura:** Progressive Web App (PWA) diseñada con enfoque *Mobile-first*.
